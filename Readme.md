# AI 활용 포트폴리오 제출 안내

> AI 활용 능력을 확인하기 위한 포트폴리오입니다.  
> 아래 7개 항목 중 **2개 이상**을 포함하여 작성 부탁드립니다.

## ⚠️ 필수 준수 사항

### 🔒 보안 준수
- 이전 근무지의 **기밀 데이터**나 **소스 코드**가 포함되지 않도록 주의
- **공개 데이터셋** 또는 **Mock-up 데이터** 활용

### 📊 기여도 명시
- 프로젝트 진행 시 본인의 역할과 생성형 AI 도구 활용 범위를 구체적으로 명시
- 예: `아키텍처 설계 100%, 코드 구현 50% + AI 보조 50%`

### 🔍 문제 해결 과정
- 단순한 결과물보다는 **데이터 전처리**, **모델 선정 이유**, **트러블 슈팅 과정**이 포함된 문서 선호

### ▶️ 실행 가능성
- 코드를 확인할 수 있는 **GitHub 링크** 또는 **데모 URL** 첨부



## 📋 포함 항목 (2개 이상 선택)

### 1. AI를 활용해 해결한 문제 또는 자동화 사례

**포함 내용:**
- 어떤 상황/문제였는지
- AI를 어떻게 활용했는지
- 결과가 어떻게 개선되었는지
- 전/후 비교 또는 개선 수치

**작성 예시:**
```markdown
## 문제 상황
- 매주 100개 이상의 고객 문의를 수동으로 분류하는 데 평균 2시간 소요

## AI 활용 방법
- Claude API를 활용한 자동 분류 시스템 구축
- 프롬프트 엔지니어링으로 카테고리 분류 정확도 향상

## 개선 결과
- 처리 시간: 2시간 → 10분 (92% 단축)
- 분류 정확도: 95%
```

---

### 2. 실제 사용한 프롬프트 또는 프롬프트 템플릿

**포함 내용:**
- 주요 프롬프트 예시
- 프롬프트 구조
- 프롬프트 개선 과정 (초안 → 수정)

**작성 예시:**
```markdown
## 프롬프트 템플릿

### 초안
"이 코드를 리팩토링해줘"

### 개선 후
"""
다음 Python 코드를 리팩토링해주세요:
- 목표: 가독성 향상 및 성능 최적화
- 제약사항: Python 3.9+ 문법 사용, 타입 힌트 포함
- 스타일: PEP 8 준수

[코드]
"""

## 개선 효과
- 응답 정확도 70% → 95%
- 원하는 형식의 답변 획득률 상승
```

### 3. AI로 생성한 초기 산출물 vs 본인이 수정한 최종 산출물

**포함 내용:**
- 코드 생성 후 리팩토링 전/후
- 자동화 스크립트 초안 vs 실제 제공한 코드
- AI가 틀리게 만든 부분과 수정한 과정

**작성 예시:**
```markdown
## AI 생성 초안
[AI가 생성한 코드 스크린샷 또는 코드 블록]

## 문제점
1. 에러 핸들링 부재
2. 메모리 누수 가능성
3. 비효율적인 알고리즘 사용

## 최종 수정본
[수정 후 코드]

## 주요 수정 사항
- try-except 구문 추가
- 컨텍스트 매니저 활용
- O(n²) → O(n log n) 최적화
```

### 5. AI의 한계를 겪고 해결한 경험

**포함 내용:**
- 환각(hallucination) 문제
- 컨텍스트 부족 문제
- 잘못된 구조/버그 생성
- 이를 어떻게 검증/조정했는지

**작성 예시:**
```markdown
## 문제 상황
Claude가 존재하지 않는 라이브러리 함수를 제안

## 해결 과정
1. AI 응답에 대한 검증 프로세스 구축
2. 공식 문서 자동 참조 시스템 추가
3. 테스트 코드 자동 생성 및 실행

## 결과
- 환각 문제 발생률 80% 감소
- 신뢰도 높은 코드 생성
```

### 7. LLM 기반 기능 개발 경험

**포함 내용:**
- AI API (OpenAI, Google, AWS, Naver 등) 연동 경험

- 유저 입력 처리 / 토큰 제어 / 프롬프트 체인 구축
- 비용 통제 경험

**작성 예시:**
```markdown
## 프로젝트: 사내 문서 검색 챗봇

### 기술 스택
- LLM: OpenAI GPT-4 API
- Vector DB: Pinecone
- Framework: LangChain

### 주요 구현 사항

#### 1. RAG 검색 구현
# 문서 임베딩 및 저장
[코드]

# 검색 및 답변 생성
[코드]

#### 2. 토큰 제어
- 입력 토큰 제한: 최대 3000 토큰
- 응답 길이 제어: max_tokens=500

#### 3. 프롬프트 체인

#### 4. 비용 통제
- 월 예산: $200
- 캐싱 전략으로 API 호출 30% 감소

### 성과
- 문서 검색 시간: 5분 → 30초
- 사용자 만족도: 4.5/5.0
- ROI: 업무 시간 절감으로 월 $2,000 상당 효과

```
